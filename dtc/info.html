How it works

Messaging workflow

  1. Customer texts your toll‑free number (reply‑only support by default).
  2. ANEMLL DTC AI parses intent and responds with guidance or status.
  3. Optional: if a user opts into alerts, we send non‑marketing status updates (≤4 msgs/mo).
  4. At any time: STOP to unsubscribe, HELP for assistance.

Direct‑to‑Cell readiness

  • Short prompts & responses for low SNR links
  • Idempotent flows to survive message loss
  • Adaptive retry windows for satellite latency
  • No app install—SMS only
